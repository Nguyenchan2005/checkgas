# Gas Monitor

Flutter application for monitoring gas levels over MQTT and showing alerts on the device.

## Requirements

- Flutter stable
- Dart SDK bundled with Flutter
- Android Studio or VS Code with Flutter support

## Run locally

```bash
flutter pub get
flutter run
```

## Notes

- Generated folders and local machine config are intentionally excluded from version control.
- If platform files are missing after clone, run `flutter pub get` and then `flutter run` to regenerate what Flutter needs.
